# APEX COACH Landing

Next.js landing page for `apexcoach.pt`, positioned around a mobile-first coaching app with a premium web companion for coaches.

## Stack

- Next.js App Router
- React 19
- Tailwind CSS
- Ready for GitHub + Vercel

## Local development

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Production build

```bash
npm run build
npm run start
```

## Deploy to GitHub and Vercel

1. Create a new GitHub repository.
2. Add the remote:

```bash
git remote add origin https://github.com/<your-user>/<your-repo>.git
```

3. Commit and push:

```bash
git add .
git commit -m "Initial Next.js landing for apexcoach.pt"
git branch -M main
git push -u origin main
```

4. Import the repository into Vercel.
5. Set the production domain to `apexcoach.pt`.
6. Use preview deployments for every PR before publishing changes.

## Current positioning

- The mobile app remains the core product for coaching in the field.
- The web experience is presented as the premium companion for planning, management, and wider operational visibility.
